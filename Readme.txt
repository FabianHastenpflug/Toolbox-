Readme

Anleitung:

    - Allgemein
        - Statt Button klicken, kann auch ENTER gedrückt werden.
        - Die Page ist resposive und explizit für die Browser-Zoomfunktion optimiert. 
        - Mit STRG + Mausrad oder ggf. Pinch&Zoom kann eine Kachel bildschirmfüllend angezeigt werden.
        - Statt "," immer "." verwenden, falls Kommazahlen nötig sind (außer beim Calcer++, da ist es egal).

    - Timer++
        - Mit "Stoppuhr" Stopuhr starten.
        - Eingabefelder füllen und mit "Timer" Timer starten.
        - Mit "Stop" Uhr stoppen und Wert zwischenspeichern.
        - Der rote Balken visualisiert die verbleibende Zeit beim Timer.

    - Noter++ 
        - Speichern: Text eingeben und ENTER.
        - 1x auf eine Zeile klicken um sie durchzustreichen.
        - 2x auf eine Zeile klicken um sie zu löschen.
        - Anzahl an Punkten ist nur limitiert durch die Größe der Box.

    - Clickcounter++
        - Irgendwo in die Box klicken, um zu Zählen.
        - Optional Startwert in Eingabefeld eingeben.
        - Einen Zählschritt zurück mit "Back".

    - Dicer++
        - Ohne Eingabe "Go!" drücken, um eine Zahl zwische 1 und 6 zu würfeln.
        - Optional Mindest- und Maximalwert, sowie Anzahl der gewünschten Zahlen eingeben.
        - Beliebig oft "Go!" klicken (oder ENTER) um erneut zu "würfeln".
        - Es werden keine Kommastellen ausgegeben.
    
    - Calcer++
        - Korrekte Rechenformel (z.B. "(5 + 2 - 1 * (4 - 7)) / 15") eigeben.
        - Es werden bis zu 3 rechnungen gespeichert, danach jeweils die älteste verdrängt.
        - Falls die Rechnung länger als 16 Zeichen ist, wird sie gekürzt und die Anzahl der fehlenden Zeichen angezeigt.

    - Color++
        - "Change" ohne vorherige Eingabe gibt der ersten Box einen zufälligen Hintergrundwert.
        - Den darauf folgenden Boxen und den Input- sowie Buttonelementen jeweils Abstufungen davon.
        - "Change" kann beliebig oft geklickt werden (alternativ auch ENTER, wenn Cursor im Eingabefeld).
        - In das Eingabefeld kann ein RGB-Hexcode eingefügt werden (egal ob mit oder ohne '#').
        - Alternativ die Dezimal-RGB-Werte, jeweils mit Leerzeichen getrennt (rrr, ggg, bbb)
        - Die aktuellen RGB-Werte werden 1. als String als Dezimalzahlen angezeigt
        - 2. visualisiert als RGB-Bars.
        - Die RGB-Bars lassen sich durch einen Klick ausblenden.
        - (Color++ kann auch als HEX-DEZ-Konverter für zweistellige Zahlen  missbraucht werden).

    - BMI--
        - Der BMI-Rechner rechnet BMI.
        - Für weitere Informationen fragen Sie Ihren Arzt oder Apotheker.

    - Namer++
        - Beliebig viele Wörter eingeben und speichern.
        - mit "Go!" wird ein beliebiges Wort ausgelost.
        - Es werden auch Wörter berücksichtigt, die aus Platzgründen nicht mehr angezeigt werden könne.

    - River++
        - In die Box klicken, um einen zufälligen Buchstaben (Umlaute ausgenommen) anzuzeigen.
        - Verwendbar z. b. für Stadt-Land-Fluss

    - Ipsumer++
        - Einen beliebig langen Lorem-Isum-Dummytext generieren.
        - Anzahl der gewünschten Zeichen eingeben und bestätigen.
        - Der Test befindet sich jetzt in der Zwischenablage und kann an jeder beliebigen Stelle eingefügt werden.

    - Wordcounter++
        - Text in das Eingabefeld einfügen und bestätigen.
        - Textanalyse erscheint.
    
    - Threer++
        - Ein 3-Satz-Rechner.
        - Als Parameter gibt es Gesamtmenge, Teilmenge und Prozentsatz
        - Es müssen immer jeweils zwei der Parameter eingetippt werden um den dritten zu berechnen.

    - Lettercounter
        - Text in das Eingabefeld einfügen und bestätigen.
        - Textanalyse erscheint: Buchstaben werden alphabetisch sortiert und deren Häufigkeit wird angezeigt.
        - Ausführlichere und übersichtlichere Darstellung der Analyse in der Browser-Konsole (F12 drücken).
